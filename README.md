# vue

## 项目设置
```
npm install
```

## 安装组件
```
npm install axios
npm install element-ui -S
npm install --save screenfull
npm install vue-router
npm install vuex --save
```

## 后台tomcat服务器地址
```
http://localhost:8080/shop/
```

## 前台vue服务器地址
```
http://localhost:8010/
```

## 运行
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
